### status code
인증실패 : 401 (HttpStatus.UNAUTHORIZED)

가능 : 204 (HttpStatus.NO_CONTENT)

유효성 검증 불합격 : 400 (HttpStatus.BAD_REQUEST)

비정상 동작 : 422 (HttpStatus.UNPROCESSABLE_ENTITY)
