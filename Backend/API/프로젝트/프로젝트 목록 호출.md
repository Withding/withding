# 프로젝트 목록 호출
### API명 : `/projects`

### method : GET

### comment :

프로젝트 목록 호출하는 API. 아래의 예시는 

{state}에는 호출하려는 프로젝트들의 상태(기본값 "진헹중"),

{page}는 호출하고자 하는 페이지 번호(기본값 1),

{cursor}는 호출을 시작할 번호(기본값 0),

{count}는 한 페이지당 호출할 글의 갯수를 넣어준다.(기본값 5)
